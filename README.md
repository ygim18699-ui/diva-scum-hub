# DIVA SCUM HUB v2

SCUM server hub with multilingual UI, server listings, settings analysis, and community board.

## Local run
```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000

## Render
- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app`
